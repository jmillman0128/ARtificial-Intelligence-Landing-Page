Activate Your Experience

Link your OpenAI API key to your headset in seconds.

  1. Put on your headset and find the 6-digit code shown on screen
  2. Get your OpenAI API key from platform.openai.com
  3. Enter the code and your key(s) below — your VR experience will start automatically
Where do I get an OpenAI API key?

Create a free account at platform.openai.com, go to API Keys in the sidebar, and click + Create new secret key. Your key starts with sk-.

For Google TTS keys (starts with AIza), visit console.cloud.google.com and enable the Text-to-Speech API.

For ElevenLabs keys (32-character hex), find yours at elevenlabs.io under Profile → API Key.

Your key is sent directly to your headset over an encrypted connection. It is never stored on this website or any server.
